Covid19 Analysis Project

SQL

I analyzed global COVID-19 data from "Our World in Data" to uncover key insights using SQL. This project involved skills such as Joins, CTEs, Temp Tables, and Window Functions to examine critical metrics like infection rates, death rates, and vaccination coverage across different countries. Key findings included the likelihood of mortality, countries with the highest infection and death rates, and global vaccination progress. Through this analysis, I aimed to demonstrate the power of data in understanding and responding to a global health crisis.
